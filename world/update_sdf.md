To update the model.sd, navigate to `/PX4-Autopilot/Tools/simulation/gz/models/x500`

Open up `model.sdf` with a text editor and paste the sdf code from `drone_modified.sdf` into the document and then save (ctrl + s)

Likewise, to modify the default world. Navigate to `/PX4-Autopilot/Tools/simulation/gz/worlds` and edit `default.sdf` to have `world_modified.sdf`
